四. 注册中心Eureka、Zookeeper、Consul的异同点

组件名 语言 CAP 服务监控检查 对外暴露接口 Springcloud集成
Eureka Java AP 可配支持 HTTP 已集成
Consul Go CP 支持 HTTP/DNS 已集成
Zookeeper Java CP 支持 客户端 已集成


最多同时满足两个

  • C：Consistency （强一致性）
  • A：Available （可用性）
  • P：Partition tolerance （分区容错性）
